# README

## Описание проекта

Этот проект представляет собой приложение на Kotlin, которое использует AI-агентов для обработки запросов и выполнения различных задач. Он включает в себя функции работы с документами, встроенные системы чатов и взаимодействия с API. Проект прошёл обновления и улучшения, чтобы повысить его производительность и функциональность.

## Структура проекта

- **app/**: Основная директория приложения.
  - **src/main/assets/**: Здесь находятся все ресурсы, используемые в приложении, включая документацию и заметки.
  - **src/main/java/com/example/aichallenge/**: Основной код приложения, включая различные классы и сервисы.
  - **src/main/res/**: Ресурсы, такие как цвета, строки и темы.
- **mcp-server/**: Серверная часть проекта, ответственная за обработку бизнес-логики и взаимодействие с данными.
  - **src/main/kotlin/com/example/mcp/**: Код серверной части, включая сервисы и модели.

## Установка и запуск

1. Клонируйте репозиторий на свой локальный компьютер.
2. Соберите проект с помощью Gradle.
3. Запустите приложение на Android Studio или запустите серверную часть с помощью IntelliJ IDEA.

## Примечания

- Обратите внимание на изменения в проекте, так как новые файлы и функции могут быть добавлены.
- Этот README документ может быть изменен в процессе работы над проектом.
- В последние обновления вошли улучшения производительности и исправления ошибок, добавлена дополнительная функциональность в классы и сервисы.
